# 宝宝速记

#### 介绍
婴儿日常行为记录，比如喂奶，辅食，便便，睡觉等

#### 软件架构

1. 本项目采用小程序的云开发，代码只是给出了业务代码，导入IDE后并不能直接运行
1. 采用vant weapp 地址:https://youzan.github.io/vant-weapp/#/intro



#### 安装教程

1.  下载代码
2.  自己先创建个云开发项目
3.  把相关的代码页面拷贝到刚才自己创建的小程序项目下
